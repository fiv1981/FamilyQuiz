# FamilyQuiz

Web app tipo trivial pensada para usarse en una pantalla horizontal de coche.

## MVP actual

- Selector de dificultad: Niño, Fácil, Medio, Difícil
- Preguntas grandes
- Respuestas grandes
- Puntuación simple
- Diseño panorámico y táctil

## Próximos pasos

- Más preguntas por nivel
- Categorías
- Sonidos
- Modo 2 jugadores
- Persistencia local
- Modo offline/PWA
