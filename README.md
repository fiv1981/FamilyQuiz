# FamilyQuiz

FamilyQuiz es una web app tipo trivial pensada para usarse en una pantalla horizontal de coche, con botones grandes, tipografía enorme y navegación muy simple.

## Incluye

- 4 dificultades: Niño, Fácil, Medio y Difícil
- Categorías: General, Geografía, Ciencia, Historia, Cine y TV, Deporte, Música y Niños
- Interfaz panorámica y táctil
- Progreso de partida y puntuación
- PWA instalable
- Funciona offline con service worker básico
- Banco amplio de preguntas precargadas

## Despliegue

No necesita build.

### Cloudflare Pages
- Framework preset: `None`
- Build command: vacío
- Build output directory: `/`
